## üìú Description

Bitscoper_SSTV_Icecast_Broadcaster is a user-friendly application that allows you to transmit images using 19 Slow Scan Television (SSTV) modes. It operates with Voice Operated Exchange (VOX) functionality through Icecast in Ogg Opus format. The application creates dynamic mount points for each transmission. This project is perfect for anyone interested in broadcasting images over the internet.
